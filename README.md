# Basic configuration
OS:        arch linux

Terminal:  urxvt

Editor:    neovim

Server:    xorg

WM:        bspwm

Hotkeys:   sxhkd

Toolbar:   polybar

# Installation
**Prerequisite:** Go through the official Arch Installation Guide and install git.

**Install:** Download the repo and run the install script.
```
$ git clone https://github.com/tossenxD/OS-configs
$ cd OS-configs/
$ chmod +x install.sh
$ ./install.sh
```
