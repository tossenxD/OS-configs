# Basic configuration
OS:        arch linux

Terminal:  urxvt

Editor:    neovim

Server:    Xorg

WM:        bspwm

Hotkeys:   sxhkd

Toolbar:   polybar

# Before installing
Go through the official Arch Installation guide. Furthermore install git. Then:
```
$ git clone https://github.com/tossenxD/OS-configs
$ cd OS-configs/
$ chmod +x install.sh
$ ./install.sh
```
